# state_run_media

Automatically publish events upon state transitions. All you have to do is subscribe.

## Installation

Add this line to your application's Gemfile:

    gem 'state_run_media'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install state_run_media
